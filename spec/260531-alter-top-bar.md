# DESCRIPTION

## 需求

1. 修改博客页面顶部的导航栏的内容以及逻辑；

2. 调整首页展示文章的样式和逻辑。

3. 修复暗色模式下，翻页存在的页面闪烁问题。

## 背景

- 我认为导航栏陈列所有标签的逻辑不好，这样子标签一但多起来就会非常臃肿。

## 目标

1. 按照我的要求，修改指定内容的代码。

# DESIGN

1. 要将顶部导航栏展示的标签，修改为分“首页”、“归档”、“分类”、“友链”这四个板块；

2. “归档”将会按照时间的先后顺序来展示文章，时间越新的排得越前，展示的内容只要文章标题、作者名字和文章更新时间（显示月份和日期即可），然后按年份分成不同的部分，例如2024年一个块里面有月份日期的、2025年又是一个块；

3. 然后我希望首页展示文章，要改成圆角矩形形式展示，一行就一个文章块，文章块两侧可以留空，然后按时间先后顺序展示，一页展示5篇文章左右；

4. 将暗色模式下，翻页闪烁问题解决。
